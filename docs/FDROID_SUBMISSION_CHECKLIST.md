# F-Droid Submission Checklist

## Source and licensing

- `[ ]` публичный source repo
- `[ ]` свободная лицензия в корне репы
- `[ ]` понятные release tags
- `[ ]` без мутных prebuilt blobs

## Build and reproducibility

- `[ ]` documented release build steps
- `[ ]` reproducibility sanity pass
- `[ ]` signed release artifact for manual checks
- `[ ]` dependencies reviewed for non-free surprises

## Metadata

- `[ ]` title
- `[ ]` short / full description
- `[ ]` screenshots
- `[ ]` icon / feature media as needed
- `[ ]` changelog
- `[ ]` source links

## SqueezeClip-specific checks

- `[ ]` убрать debug signing из release story
- `[ ]` объяснить или убрать network-related permissions
- `[ ]` privacy policy и behavior совпадают
- `[ ]` README и docs не врут про текущие возможности
